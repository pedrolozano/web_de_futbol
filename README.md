# web_de_futbol

Pequeño sitio estático para mostrar información de un torneo de fútbol.

Cómo ejecutar localmente:

- Usando Python (desde la carpeta del proyecto):

```powershell
python -m http.server 8000
# Abrir http://localhost:8000
```

Estructura principal:

- `index.html` — contenido principal
- `styles.css` — estilos
- `script.js` — comportamiento cliente
